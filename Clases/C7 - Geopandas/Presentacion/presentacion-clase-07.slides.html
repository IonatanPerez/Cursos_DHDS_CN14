GIS (Sistemas de Informacin Geogrfica)

Sistemas de Información Geográfica (SIG)¶

Se definen como sistemas que permiten almacenar datos geoespaciales para su consulta, manipulacin y representacin. La representacin de estos datos es el campo de estudio de la Cartografa. A partir del trabajo con los SIG, naci una nueva disciplina cientfica, conocida como Ciencia de la Informacin Geogrfica.

El sistema permite separar la informacin en diferentes capas temticas, las almacena independientemente, y luego se pueden presentar todas juntas.

Datos Geoespaciales¶

Los Datos Geoespaciales, tambin llamados informacin geogrfica, geodatos o simplemente datos espaciales, son datos georreferenciados. La georreferenciacin es la tcnica de posicionamiento espacial de una entidad en una localizacin geogrfica nica y bien definida en un sistema de coordenadas.

Los geodatos poseen informacin implcita (cantidad de habitantes, una referencia catastral, una direccin postal, etc.) o explcita (coordenadas de ubicacin obtenidas a partir de datos capturados mediante GPS, etc.).

Sistema de coordenadas geográficas¶

Son los sistemas que permiten referenciar un objeto en una posicin espacial determinada. El sistema ms usual es el WGS 84, que usa las medidas latitud y longitud.

Si consideramos a la Tierra como una esfera (o esferoide), existen dos lneas fundamentales: el meridiano de Greenwich, que va de un polo a otro, y el Ecuador, que cruza al eje de rotacin por el centro de la esfera. Las lneas paralelas al Ecuador se denominan latitud, y las lineas paralelas a Greenwich, longitud.

La latitud se indica con valores positivos entre 0 y 90 para las lneas al norte del Ecuador, y con valores negativos entre 0 y -90 para las que estn al sur. Otra forma de nombrarlas es usando siempre un valor positivo entre 0 y 90, y agregando si corresponde al Norte o al Sur.

La longitud se indica en forma similar, pero con valores positivos entre 0 y 180 para las lneas al este de Greenwich, y con valores negativos entre 0 y -180 para las que estn al oeste. O bien siempre un valor positivo entre 0 y 180, e indicando este u oeste.

Tipos de Datos Geoespaciales

Tipos de Datos Geoespaciales¶

Existen dos tipos de datos geoespaciales en los Sistemas de Informacin Geogrfica: raster y vectorial.

Tipos de Datos Raster¶

El tipo de datos raster es lo que consideramos imagenes digitales, o mapas. Se representan mediante mallas o cuadriculas; divide el espacio en celdas regulares donde cada una de ellas representa un nico valor. Similar a la fotografa digital donde el pxel es la unidad menor de informacin de una imagen, y una combinacin de estos pxeles crea una imagen.

Este tipo de datos se centra en las propiedades del espacio, por ejemplo, elevaciones sobre un terreno, nivel de luz, variedad de cultivos. No siendo tan precisos en la localizacion en el espacio.

Mapa y mapa pixelado

Tipos de Datos Vectorial¶

El tipo de datos vectorial representan objetos que provienen de diversas fuentes: localidades, negocios, barrios, rutas, nivel de contaminacin, etc. Son datos discretos, es decir, con limites definidos. Y con una localizacion precisa sobre el espacio.

Para modelar los objetos se utilizan tres formas: el punto, la lnea y el polgono.

Tipos de Datos Vectorial¶

  • Los puntos se utilizan para las entidades geogrficas que mejor pueden ser expresadas por un nico punto de referencia. Por ejemplo, estaciones de subte, edificios, localizaciones de los pozos, puntos de inters.
  • Las lneas unidimensionales o polilneas son usadas para rasgos lineales como ros, caminos, ferrocarriles, lneas topogrficas o curvas de nivel. En los elementos lineales puede medirse la distancia.
  • Los polgonos se utilizan para representar elementos geogrficos que cubren un rea particular de la superficie de la tierra. Por ejemplo, lagos, lmites de parques naturales, yacimientos, provincias. Con ellos se pueden medir el permetro y el rea.

Archivos de Datos Geoespaciales¶

En general, se denomina formato SIG a los formatos de archivos con datos geoespaciales.

Los datos geoespaciales de tipo raster se almacenan en los tipos de archivos que conocemos para almacenar imgenes. Por ejemplo, los formatos TIFF, JPEG, PNG, etc.

Los datos geoespaciales de tipo vectorial ms comunes son:

  • SHP (Shapefile). El archivo shapefile es el tipo de archivo geoespacial ms comn. Todos los formatos comerciales y de cdigo abierto aceptan shapefile como formatos SIG. Se ha convertido en el estndar de la industria.

  • CSV / GeoCSV. CSV (Comma-separated values) representa datos en forma de tabla separados por comas. Es usual encontrarlos en publicaciones de organismos, por ejemplo https://data.buenosaires.gob.ar/dataset. La ventaja es que ocupan poco espacio y es fcil compartirlos. GeoCSV es el mismo formato CSV con una extensin de geometra espacial.

Existen otros tipos de archivos, por ejemplo: GeoJSON (Javascript Object Notation), KMZ/KML (Keyhole Markup Language), OSM (OpenStreetMap), PostGIS + PostgreSQL, etc.

GeoPandas

GeoPandas¶

GeoPandas es una biblioteca que permite trabajar con datos geoespaciales en Python. Para ello, usa las facilidades de otras bibliotecas de Python.

  • Extiende la funcionalidad de pandas. Es decir que se pueden realizar todas las operaciones que se aplican a las Series y DataFrame.

  • Provee una interfaz con la libreria matplotlib para generar mapas. Las figuras geomtricas se pueden mapear facilmente con el mtodo plot().

  • Permite operaciones sobre los datos geomtricos que representan a los datos espaciales (puntos, lneas y polgonos). Para ello hace uso de la libreria shapely.

  • Facilita el acceso a datos en mltiples formatos de archivos, extendiendo las operaciones que brinda la librera fiona.

  • Realiza proyecciones, es decir, permite usar distintos tipos de coordenadas para referenciar la posicin geogrfica de los datos geoespaciales.

GeoSeries y GeoDataFrame¶

As como Pandas tiene como estructura de datos a las Series y Dataframe, Geopandas tiene algo similar llamado Geoseries y GeoDataFrame.

La diferencia entre ellos es que tanto GeoSeries como GeoDataFrame, deben contener al menos una columna con tipos de datos geoespaciales. Esta columna se la llama por default geometry.

La columna geometry contiene una forma geomtrica que representa la posicin espacial del objeto. Por ejemplo, la posicion espacial de un negocio - sus coordenadas- se expresan mediante la forma geomtrica punto.

Las formas geomtricas son:

  • POINT. Un punto
  • LINESTRING. Una lnea
  • POLYGON. Una superficie

Tambin se pueden representar colecciones de formas:

  • MULTIPOINT. Varios puntos
  • MULTILINESTRING. Varias lneas
  • MULTIPOLYGON. Varias superficies

En Python¶

Vamos a trabajar con algunos datos geoespaciales con GeoPandas.

  • Puntos, que representan un lugar determinado. Como tienen cero dimensiones no se puede medir la longitud ni el rea. Un punto puede representar datos abstractos, como una ciudad, en una escala pequea. A medida que se hace zoom, y se agranda la escala, se transforma en un rea, un polgono.

  • Lneas, que pueden representar ros, senderos, calles, etc. Como tienen una dimensin, se puede medir su longitud. Tienen un punto inicial y un punto final. Tambin se usan para representar arcos.

  • Polgonos, se utilizan para representar reas tales como el lmite de una ciudad (en un mapa a gran escala), un lago o un bosque. Las caractersticas del polgono son bidimensionales y, por lo tanto, se pueden usar para medir el rea y el permetro de una caracterstica geogrfica.
